# Memory Match

Memory Match is a browser-based 12-card matching game implemented with static HTML, CSS, and JavaScript.

## Files

- `memory-match.html` - Play screen, status message, board container, reset button, and navigation link.
- `memory-match.js` - Deck creation, shuffling, card rendering, match detection, move counting, and reset behavior.
- `memory-match.css` - Responsive card grid, flip animation, matched state, and game page styling.

## Gameplay notes

- The deck contains 6 pairs: apple, banana, orange, bicycle, basketball, and briefcase.
- Cards use inline SVG illustrations generated by `memory-match.js`.
- Two selected cards stay matched when their pair IDs match.
- Mismatched cards flip back after an 800 ms delay while the board is locked.
- The game reports the number of moves used after all pairs are matched.

## Testing

Memory Match behavior is covered by the Node test harness in `../test/games.test.js`.
