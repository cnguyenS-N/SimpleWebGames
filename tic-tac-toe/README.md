# Tic-Tac-Toe assets

This directory contains the stylesheet for the Tic-Tac-Toe experience:

- `tic-tac-toe.css` - visual styles shared by the Tic-Tac-Toe landing page and play screen.

The rest of the Tic-Tac-Toe files currently live outside this asset directory:

- `../tic-tac-toe.html` - landing page with the play call to action.
- `../game/game.html` - 3x3 play screen markup.
- `../game/game.js` - win detection, tie detection, board rendering, and computer move selection.
