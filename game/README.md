# Game Directory

This directory currently holds the shared game picker markup and the Tic-Tac-Toe play screen.

## Files

- `index.html` - Browser games picker page.
- `game.html` - Tic-Tac-Toe play screen with the 3x3 board, status text, and reset button.
- `game.js` - Tic-Tac-Toe game logic, including winner checks, tie checks, computer moves, rendering, and reset handling.

## Development notes

- `game.html` is part of the Tic-Tac-Toe experience even though the latest landing page and stylesheet live in `../tic-tac-toe/`.
- Keep the game picker links and Tic-Tac-Toe asset paths synchronized with the directory layout whenever files are moved.
